# BaseLoader\#

_class _langchain\_core.document\_loaders.base.BaseLoader[\[source\]](https://python.langchain.com/api_reference/_modules/langchain_core/document_loaders/base.html#BaseLoader)\#     

Interface for Document Loader.

Implementations should implement the lazy-loading method using generators to avoid loading all Documents into memory at once.

load is provided just for user convenience and should not be overridden.
