Load Microsoft PowerPoint files using Unstructured.

Works with both .ppt and .pptx files. You can run the loader in one of two modes: ‚Äúsingle‚Äù and ‚Äúelements‚Äù. If you use ‚Äúsingle‚Äù mode, the document will be returned as a single langchain Document object. If you use ‚Äúelements‚Äù mode, the unstructured library will split the document into elements such as Title and NarrativeText. You can pass in additional unstructured kwargs after mode to apply different unstructured settings.
