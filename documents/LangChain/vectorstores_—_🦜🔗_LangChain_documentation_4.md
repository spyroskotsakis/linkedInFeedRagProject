# `vectorstores`\#

**Vector store** stores embedded data and performs vector search.

One of the most common ways to store and search over unstructured data is to embed it and store the resulting embedding vectors, and then query the store and retrieve the data that are â€˜most similarâ€™ to the embedded query.

**Class hierarchy:**               VectorStore --> <name>  # Examples: Annoy, FAISS, Milvus          BaseRetriever --> VectorStoreRetriever --> <name>Retriever  # Example: VespaRetriever     

**Main helpers:**               Embeddings, Document
